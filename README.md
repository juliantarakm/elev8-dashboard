# Elev8 Dashboard

Property management dashboard for Bali vacation rentals.

## Quick Start

```bash [Terminal]
cd Dashboard
pnpm install
pnpm run dev
```

## App Settings

Change settings in `app.config.ts`. Clear cookie 'app_settings' first if needed.

## Tech Stack

- Nuxt 4
- Vue 3
- shadcn-vue
- Tailwind CSS 4
